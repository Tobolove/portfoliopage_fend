Coffee Shop Project Dynamic Online Menu

Education

This project involved engineering a full-stack, digitally-enabled cafe experience, centered around a secure and dynamic drink menu. The core challenge was to build a sophisticated API with a multi-layered security architecture to manage access for different user roles—from public customers to internal staff.

Zero-Trust Security Architecture

I designed and implemented a robust security model using Auth0 for identity management. The platform features a fine-grained permission system based on user roles (Public, Barista, Manager), enforced by validating JSON Web Tokens (JWTs) on the backend. Each API endpoint is protected by specific permission requirements, ensuring secure access control.

Key Features

  • Zero-trust security architecture with Auth0 identity management
  • Role-based access control (RBAC) for Public, Barista, and Manager roles
  • JWT-based authentication and authorization for all API endpoints
  • Comprehensive RESTful API for menu and drink management
  • Ionic frontend providing intuitive user interface
  • Real-time menu updates
  • Scalable database design for menu items
