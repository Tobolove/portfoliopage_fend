Fyyur: Musical Venue & Artist Booking Site

Education

Fyyur is a full-stack web application that serves as a booking site for musical venues and artists, facilitating the discovery of new talent and locations. The platform replaces static mock data with a dynamic, database-driven backend to create a fully persistent and interactive experience.

Database-Driven Platform

I transitioned the application from using mock data to a robust and persistent system by connecting it to a PostgreSQL database. Using SQLAlchemy ORM, I designed a relational database schema with normalized models for Artists, Venues, and Shows, ensuring data integrity and optimal performance.

Key Features

  • Dynamic venue and artist discovery platform with advanced search capabilities
  • Real-time show booking system with availability tracking
  • Comprehensive artist and venue profile management
  • Interactive dashboard for managing bookings and performances
  • Normalized relational database design for optimal data relationships
  • Full CRUD operations for venues, artists, and shows
  • Advanced filtering and search functionality across all entities

Technology Stack

Backend

Python, Flask, SQLAlchemy

Database

PostgreSQL

Migrations

Flask-Migrate, Alembic

Frontend

HTML, CSS, JavaScript, Bootstrap
